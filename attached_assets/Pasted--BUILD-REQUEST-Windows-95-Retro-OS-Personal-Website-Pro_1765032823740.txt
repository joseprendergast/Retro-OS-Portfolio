## Window System Requirements

### Window Components (All windows must have)

**Title Bar:**
- Application icon (left)
- Window title text (centered or left-aligned)
- Minimize button (underscore icon)
- Maximize/Restore button (square icon)
- Close button (X icon)
- Gradient: Blue (active) or Gray (inactive)

**Menu Bar:** (Below title bar)
- File, Edit, View, Help menus
- Can be functional or decorative based on app

**Content Area:**
- Scrollable if content exceeds window size
- White or app-specific background

**Status Bar:** (Bottom, optional)
- Shows context info or instructions

**Borders:**
- Resizable borders on all sides
- Corner resize handles (8-directional)
- 3D raised border effect

### Window Behaviors

**Opening:**
- Fade in + scale animation
- Opens centered or cascaded if multiple windows
- Comes to front (highest z-index)

**Dragging:**
- Click and drag title bar to move
- Cursor changes to move cursor
- Window drags smoothly with mouse

**Resizing:**
- Hover on borders/corners shows resize cursor
- Click and drag to resize
- Minimum size constraints (prevent too small)
- Maximum size: screen minus taskbar

**Minimizing:**
- Animates shrinking to taskbar button
- Window hidden but tracked

**Maximizing:**
- Expands to fill screen (minus taskbar)
- Maximize button becomes Restore button
- Borders removed when maximized

**Closing:**
- Scale down + fade out animation
- Removed from taskbar
- Confirm dialog for important actions

**Focus Management:**
- Click any part of window brings to front
- Active window has blue title bar
- Inactive windows have gray title bar
- Only one active window at a time

## Visual Design Requirements

### Color Palette (Authentic Windows 95)
- **Desktop**: Teal (#008080) or wallpaper
- **Window/Button Gray**: #c0c0c0
- **Dark Gray**: #808080
- **White**: #ffffff
- **Black**: #000000
- **Active Title Bar**: Gradient blue (#000080 to #1084d0)
- **Inactive Title Bar**: Gradient gray (#808080 to #c0c0c0)
- **Highlight Blue**: #000080

### Typography
- **System Font**: MS Sans Serif or equivalent (Arial, Segoe UI as fallbacks)
- **Fixed Width**: Fixedsys or Courier New for system dialogs
- **Icon Labels**: 8pt, white text with black shadow/outline
- **Menu Text**: 8pt
- **Title Bar**: 8pt bold

### Border Styles (Critical for authenticity)
- **Raised/Outset**: Light top-left, dark bottom-right (buttons, window borders)
- **Sunken/Inset**: Dark top-left, light bottom-right (text inputs, content areas)
- **Flat**: No 3D effect (some menu items)
- **Border Width**: 2px typical
- **Colors**: 
  - Highlight: #ffffff
  - Light: #dfdfdf
  - Shadow: #808080
  - Dark Shadow: #000000

### Cursors
- **Default**: Standard arrow
- **Pointer**: Pointing hand (over links/buttons)
- **Text**: I-beam (over text)
- **Move**: Four-directional arrows (dragging windows)
- **Resize**: Directional arrows (window borders)
- **Wait**: Hourglass (loading states)

### Icons Style
- Pixelated 32x32 or 48x48
- Limited color palette (256 colors max)
- Slight perspective/3D effect
- Classic Windows 95 aesthetic

## Interaction Details

### Mouse Interactions
- **Hover**: Highlight state (buttons, menu items, icons)
- **Click**: Pressed state (buttons depress visually)
- **Double-Click**: Opens items (icons, files)
- **Right-Click**: Context menu
- **Drag**: Move items, select multiple
- **Drag Window**: Move windows by title bar
- **Resize**: Drag window borders/corners

### Keyboard Interactions (Phase 1 Basic)
- **Alt + F4**: Close active window
- **Enter**: Activate selected item
- **Escape**: Close dialog/menu
- **Tab**: Navigate between elements
- **Arrow Keys**: Navigate Start Menu
- **F5**: Refresh window

### Dialog Boxes
- **Confirm Dialogs**: "Are you sure?" with Yes/No/Cancel
- **Alert Dialogs**: OK button only
- **Input Dialogs**: Text field with OK/Cancel
- **Always modal**: Blocks interaction with parent window
- **Always centered**: On screen or parent window

### Loading States
- Hourglass cursor
- "Please wait..." dialog (for longer operations)
- Progress bars (horizontal, green/blue filling)

### Window Animations
- **Open**: Fade in (0.2s) + scale from 0.9 to 1
- **Close**: Fade out (0.15s) + scale to 0.95
- **Minimize**: Shrink to taskbar button (0.3s)
- **Maximize**: Expand to fullscreen (0.2s)
- **Restore**: Return to previous size/position (0.2s)

### Button Animations
- **Hover**: Subtle highlight (instant)
- **Press**: Sunken effect (instant)
- **Release**: Return to normal (instant)

### Menu Animations
- **Open**: Slide down with fade (0.15s)
- **Submenu**: Slide right with fade (0.15s)
- **Close**: Fade out (0.1s)

### Icon Animations
- **Select**: Blue highlight background (instant)
- **Drag**: Follow cursor with slight lag/smoothing

### Keep animations subtle and fast - Windows 95 was snappy!

### Mobile (< 768px)
- **Simplified Mode**:
  - Taskbar becomes bottom navbar
  - Start Menu becomes slide-out drawer
  - Icons larger for touch (64x64)
  - Windows open fullscreen (no drag/resize)
  - Swipe between windows
- **Landscape recommended**: Show notification on portrait mode
- Desktop icons in 2-column grid

### Touch Interactions (Tablet/Mobile)
- Tap = Click
- Long press = Right-click
- Two-finger tap = Right-click (alternative)
- Pinch-to-zoom disabled
- Swipe up from taskbar = Show open windows
